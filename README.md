# filigranYokEt

Bazı resimlerde kopyalanmayı önlemek için resim arkaplanına bir filigran konur.
Bu filigranın pixel renk kodları alınarak yerine resmin orjinal pixel rengini koyarsak filigran ortadan kalkar
